# Assignment 1: Build your first CRUD API

A simple REST API built with **FastAPI** as part of the FlyRank Backend AI Engineering Internship.

The API allows users to create, retrieve, and update tasks using standard HTTP methods. It is designed as an introduction to backend development, REST APIs, request validation, HTTP status codes, and automatic API documentation with Swagger UI.

---

## Features

- RESTful API built with FastAPI
- Automatic interactive API documentation with Swagger
- In-memory task storage
- Input validation
- Proper HTTP status codes
- Error handling with informative responses

---

## Installation & Running

Clone the repository:

```bash
git clone https://github.com/trisha-s-101/flyrank-assignments.git
cd flyrank-assignments
```

Install dependencies:

```bash
pip install fastapi uvicorn
```

Run the server:

```bash
uvicorn main:app --reload
```

The API will be available at:

- API: http://localhost:8000
- Swagger UI: http://localhost:8000/docs

---

## API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/` | API information |
| GET | `/health` | Health check |
| GET | `/tasks` | Retrieve all tasks |
| GET | `/tasks/{id}` | Retrieve a task by ID |
| POST | `/tasks` | Create a new task |
| PUT | `/tasks/{id}` | Update an existing task |
|DELETE| `/tasks{id}` | Delete an existing task

---

## Example Request

```bash
curl -i -X POST http://localhost:8000/tasks \  
-H "Content-Type: application/json" \
-d '{"title":"Finish assignment"}'
```
